# HALO Console v0.7.8-local Model Selector Polish

HALO Console v0.7.8-local is a sidebar UI polish checkpoint for the local model selector.

## What changed

- Active Model and Installed Models were merged into one compact Models card.
- The card keeps the active model dropdown for choosing installed local Ollama models.
- Installed model information remains visible as a compact Quick, Daily, and Heavy summary list.
- The sidebar is shorter and less visually redundant.

## Behavior unchanged

- No model runtime behavior changed.
- No Ollama API behavior changed.
- No document behavior changed.
- No retrieval behavior changed.
- No Learning Layer behavior changed.
- No security behavior changed.

## Not added

- No cloud APIs.
- No OpenAI integration.
- No web search enablement.
- No OCR.
- No shell execution.
- No arbitrary filesystem reads.
- No secret storage.
